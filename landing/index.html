What is go-pflow?

go-pflow is a Go library for building applications with Petri nets - a mathematical model for describing concurrent systems. Instead of writing state machines by hand, you define places (states), transitions (actions), and arcs (connections). The library handles state management, concurrent execution, and even predictive analysis via ODE simulation.

Each demo below teaches specific go-pflow concepts. Start with Tic-Tac-Toe to learn the basics, then explore Coffee Shop for resource modeling, and Texas Hold'em for complex multi-player state.
