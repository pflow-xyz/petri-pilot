How It Works

This simulator uses a Petri net model where each board position is represented as a place, and moves are transitions that consume empty-cell tokens and produce player-piece tokens.

The strategic values are derived using ODE (Ordinary Differential Equation) simulation from the go-pflow library. Each candidate move is evaluated by computing:

Score = my_win - opponent_win
The center position scores highest (0.430) because it participates in 4 winning patterns. Corners score 0.316 (3 patterns), and edges score 0.218 (2 patterns). These values emerge purely from the Petri net topology, without explicit heuristics.
